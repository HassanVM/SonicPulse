## Prosjekt

Prosjektet "SonicPulse" er en enkel applikasjon som gjør det mulig å spille av musikk der brukeren kan selv velge mellom hvilken musikk brukeren ønsker å spille. Det er også forskjellige funksjoner der brukeren kan endre lyden, spille av musikk automatisk. 


## Demo

https://sonicpulse.netlify.app/

## Teknologier
- HTML
- CSS
- Javascript
- React


## Installasjon og Oppsett

1. Klone dette repositoryet: `git clone https://github.com/HassanVM/SonicPulse.git`
2. Naviger til prosjektmappen: `cd sonicpulse-main`
3. Installer avhengigheter: `npm install`
4. Start prosjektet ved å skrive npm start 
5. Dersom du ikke ble automatisk rettet til prosjektet, så åpne nettleseren og gå til `http://localhost:3000`
