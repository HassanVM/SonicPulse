## Prosjekt

Prosjektet "SonicPulse" er en enkel applikasjon som gjør det mulig å spille av musikk der brukeren kan selv velge mellom hvilken musikk brukeren ønsker å spille. Det er også forskjellige funksjoner der brukeren kan endre lyden, gjenopprette musikk automatisk. 


## Demo

https://sonicpulse.netlify.app/

## Teknologier
- HTML
- CSS
- Javascript
- React


## Installasjon

1. Klone dette repositoryet: `git clone https://github.com/HassanVM/SonicPulse.git`
2. Naviger til prosjektmappen: `cd sonicpulse`
3. Installer avhengigheter: `npm install`

## Oppsett

1. Start utviklingsserveren: `npm start`
2. Åpne nettleseren og gå til `http://localhost:3000`
